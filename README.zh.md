# 抖音视频下载器 <!-- by 梁志灿 -->
专业级抖音内容下载工具。

## 核心功能
- **视频下载**：
  - 通过分享链接下载单个视频
  - 批量下载用户主页作品（最多3000个）
  - 支持4K/高清/标清画质选择
  - 可选无水印版本下载

- **特色内容**：
  - 直播录制（含弹幕捕获）
  - 图集/图片作品下载
  - 音乐提取（MP3格式）
  - 封面图片保存

## 技术实现
- 混合下载模式（直连+代理）
- 自适应分块大小（8KB-1MB）
- 连接池技术（keep-alive）
- 自动重试机制（3次尝试）
- 元数据提取（JSON/XML）
- 文件名规范化
- 内容去重
- 哈希校验

## 性能指标
- 吞吐量：约50MB/秒（10线程）
- 并发数：最高10个同时下载
- 内存占用：平均<50MB
- CPU利用率：<30%（Intel i5）

## 支持平台
- Windows 10/11 (x64)
- macOS 12+ (ARM/Intel)
- Linux (Ubuntu 20.04+)
